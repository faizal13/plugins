Mortgage IPA Digitalising In-Principle Approval for Mortgage Lending

A fully digital, AI-powered mortgage pre-approval platform that replaces today’s manual, email-driven IPA process — enabling customers to get an In-Principle Approval in minutes, while freeing bank teams to focus on converting IPAs into mortgages.

1
The Problem

Why We Need This

👤
For the Customer

Customers face a fully manual, email-driven IPA process with no document validations, constant back-and-forth with sales, and zero real-time visibility — making mortgage pre-approval slow and frustrating.

🏦
For the Bank

Only 30% of IPAs convert to mortgages, yet each takes 45–60 min across Sales and Underwriting with manual AECB checks and no validations — effort that should go toward final conversions and Central Bank compliance.

2
Our Solution

How Mortgage IPA Solves This

A guided digital journey that replaces every manual step with automated, validated workflows.

🔍

Real-Time AECB Integration

Instant credit bureau check with customer consent — no manual keying into the AECB calculator.

✅

Built-In Document Validations

File type, size, and completeness checks at upload — eliminates back-and-forth between Sales & Credit.

⚡

Automated Eligibility Engine

Instant IPA decision based on income, liabilities, and AECB data — from days to minutes.

📱

Customer Self-Service (Web + Mobile)

Customers complete the entire IPA journey on their phone — no branch visits, no RM dependency.

🛡

Central Bank Compliance

KFS, consent management, and CBUAE mortgage lending regulations built into every step.

👥

Channel Partner Portal

Brokers, Direct Sales, and Elite Sales send invites via a dashboard — customer gets an SMS to start.

Autonomous SDLC

How Our AI Agents Work

Six purpose-built agents fire automatically at trigger points. One human gate preserves engineering judgment.

01
Story Analyzer
You paste story ID in Copilot Chat

Reads ADO story & acceptance criteria, pulls solution-design docs, classifies service, personas, state transitions, API contract, and test cases. Creates a structured GitHub Issue.

ADO MCPGitHub MCP~3–5 min
02
Coding Agent
Triggered by ai-generated label

Checks out release branch, creates feature branch, generates full implementation: Flyway migrations, JPA entities, service + state machine, controller, OpenAPI, and test suites. Raises a Draft PR.

Full stackCopilot Workspace~10–15 min
03
AI Code Review
Triggered by PR opened

Reviews diff against curated instructions. Posts inline comments: BigDecimal violations, missing AC tests, persona gaps, missing timeouts — before a human opens the PR.

GitHub ActionsInline comments~3–5 min
🧑
Human Gate
Your engineering judgment

Review generated code for business logic, address AI review comments, run tests locally, approve & merge. The only place human judgment is required.

Business logic check~20 min
05
Learning Agent
Triggered by PR merge

Extracts coding patterns from approved PRs, scores confidence, promotes to .copilot/skills/ when confidence ≥ 85%. The system gets smarter every sprint.

Self-learningGitHub Models API~2–3 min
06
ADO Sync
Triggered by push to main

Extracts ADO story ID from commit, moves story to Done, links production commit. PO sees story close automatically with full traceability.

ADO REST APIAuto-close~1 min
